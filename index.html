Progressive Enhancement Writing Assignment

The first thing that become acutely aware to me when working on this project was the difficulty writing the css soon after starting the project. It took me three attempts to settle on an acceptable markup for the first draft. But I was suprised to notice a real progressive feeling of learning from each attempt. The final attempt was considerably more refined than the first, and how better to use html classes and IDs clicked. Doing individual excersices were relatively easy to comprehend, but the project is definitely an eye opener in how to use our newly aquired toolkit. But that only came with trial and error. Even now as I work on the second draft, I'm noticing various obtimal changes to make to the css, such as removing fixed units of measurements in places where responsive units would be preferred.
